Project Report: 
---
Marks vs CGPA Predictor

1. Introduction

In this project, I worked on predicting a student's CGPA based on their subject marks.
The main aim was to understand how machine learning can be used to find relationships between marks and CGPA.

---

2. Problem Statement

Students often want to estimate their CGPA before the final results are declared. However, it is not
always easy to understand how marks in different subjects affect overall CGPA.
The problem I chose was:
Can we predict CGPA based on subject marks using a machine learning model?

---

3. Why This Problem Matters

This problem is useful for students because it gives them an idea of their expected performance.
It can also help in planning improvements if the predicted CGPA is low.
Even though the model is simple, it shows how real-world prediction systems work.

---

4. Approach

To solve this problem, I used a supervised machine learning approach. The dataset contains marks 
of four subjects: Maths, Physics, Computer Science, and English, along with their corresponding CGPA.
I used Linear Regression because:

•	It is easy to understand

•	It works well for continuous output values like CGPA

•	It is suitable for beginners

---
Steps followed:

1.	Created a dataset manually with marks and CGPA

2.	Converted data into features (X) and target (y)

3.	Split the dataset into training and testing data

4.	Trained the model using Linear Regression

5.	Took user input and predicted CGPA

---

5. Key Decisions

Some important decisions I made during the project:

•	Used a small but realistic dataset 

•	Chose Linear Regression

•	Kept the project simple to focus on understanding


---

6. Challenges Faced

•	Initially, the dataset was too small, which gave unstable predictions

•	Understanding how to properly format input for prediction

•	Setting up libraries like sklearn and pandas

I solved these by improving the dataset and testing multiple inputs.

---

7. Results

The model is able to predict CGPA reasonably well based on given marks.
Although it is not perfectly accurate, it gives a good approximation.

---

8. Learning Outcomes

From this project, I learned:

•	Basics of machine learning

•	Difference between regression and classification

•	How to train a model and make predictions

•	Importance of good data

---

9. Future Improvements

•	Use a larger and real dataset

•	Add more subjects for better accuracy

•	Build a GUI for better user interaction

•	Try advanced ML models

---

10. Conclusion

This project helped me understand how machine learning works in a practical way.
Even though it is a simple model, it shows how predictions can be made using data.
It also gave me confidence to explore more advanced ML projects in the future.
